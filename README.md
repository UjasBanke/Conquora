# Conquora

### Turn-Based Strategy Game (Dots and Boxes)

Conquora is a two-player, turn-based strategy game inspired by **Dots and Boxes.**
The project focuses on deterministic game logic, clean state management, and a polished canvas-based UI.



## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API




## Getting Started

1. Clone the repository:
 ```bash
git clone https://github.com/UjasBanke/Conquora.git
```


2. Open `index.html` in a modern browser.

No build tools or server are required.


Have fun playing!
